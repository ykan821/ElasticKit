# Contributing

## Workflow

1. Fork the repo, create a branch from master
2. Make sure all tests and static analysis pass
3. Submit a PR to master

## Reporting Issues

Please include your PHP version, Elasticsearch version, and a code snippet to reproduce the problem.

## License

By submitting code, you agree to license it under the MIT License.

# 贡献指南

## 开发流程

1. Fork 仓库，从 master 创建分支
2. 改完后跑测试和静态分析，确保全部通过
3. 提交 PR 到 master

## 报告问题

提 Issue 时请包含 PHP 版本、Elasticsearch 版本、复现步骤或代码片段。

## 许可

提交代码即表示你同意以 MIT 协议授权。
